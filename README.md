# 🔥 DoH Proxy Client
Simple DoH Proxy Tool - Submit all requests through DoH ( DNS-Over-Https )

#
# 📦 Usage
Prerequisites
  + Go 1.21+ installed.
  + Terminal access.

1. Clone the Repository
``` bash
git clone https://github.com/NIKJOO/DoH-Proxy-Client.git
cd DoH-Proxy-Client
```

2. Install necessary packages
```bash
go mod init myproject
go get github.com/miekg/dns
```

3. Run 
```bash
go run main.go
```
