# 🔥 DoH Proxy Client
Simple DoH Proxy Tool - Submit all requests through DoH ( DNS-Over-Https )

#
# 📦 Usage
Prerequisites
  + Go 1.21+ installed.
  + Terminal access.

1. Clone the Repository
<pre>    git clone https://github.com/NIKJOO/DoH-Proxy-Client.git
    cd DoH-Proxy-Client </pre>
2. Run 
<pre>    .\go run main.go
     </pre>
